# 内蒙古坍塌煤矿救援难度非常大，坍塌山体基本将底部作业面全覆盖

据内蒙古阿拉善盟盟长汇报，此次坍塌事故发生区域基本是个山谷。山体整体发生塌方后，基本上将底部作业面全部覆盖，救援难度非常大。经连夜勘察评估，专家组已制定优化救援方案，正组织救援人员和机械加紧搜救失联人员。持续关注，愿平安！（总台央视记者张腾飞）
