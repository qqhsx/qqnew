# 中国商人泰国酒店遇害案3名嫌犯落网，系被性工作者下药后与同伙谋杀

极目新闻记者 李曼英

据泰国《每日新闻》5月17日最新报道，当地时间5月16日，43岁中国商人李先生在泰国度假村被刺20余刀身亡，财物被洗劫一空。事发后，省警9区副局长亲临现场视察，并已下令调查。当地时间16日晚上10时左右，警方突击搜查，并逮捕了3名嫌疑人，包括与李先生开房的那名20岁左右的泰国女子。

经初步调查，事发当天，李先生与这位泰国年轻女子约会，该女子是一名性工作者。这名女子当天带着她的2名男友一起前往度假村，女子与李先生开了房，2名男子则入住其他房间。

![](https://inews.gtimg.com/om_bt/OTzj-Z0gKv4y0pYMMOAQm_ZzLoWydlkW_eqkTVSbuIU0cAA/1000)_报道截图（图源：泰国《每日新闻》）_

据嫌疑人交代，当天，这名年轻女子服务不好，让李先生不高兴。李先生后来中毒，随后用力扇打女子的脸。与女子同行前来度假村的男子用刀杀害了李先生，随后带上财物，驾车逃亡。

警方称，该中国男子已婚，与泰国妻子育有3个孩子。他的妻子后来到度假村看了他的遗体，由于悲伤，她没有对嫌疑人的可能身份发表观点。为了家庭的隐私，受害者妻子的全名被警方隐去了。
