

# 天津警方通报3名城管疑被捅伤：摊主持刀致3人受伤，已被刑拘

情况通报

针对互联网上出现“宁河3名城管疑被捅伤”的视频，经查，2023年8月16日7时20分许，在宁河区芦台街道一公园门前，宁河区综合执法工作人员在对摊主刘某某
(男，60岁)
占道经营行为管理过程中，刘某某持刀致3名综合执法工作人员受伤。接报警后，民警迅速赶到现场，将刘某某控制。伤者经送医院治疗，均无生命危险。目前，刘某某已被公安机关依法刑事拘留，案件正在侦办中。

公安宁河分局
