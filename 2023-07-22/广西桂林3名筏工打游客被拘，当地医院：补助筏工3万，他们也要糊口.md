

# 广西桂林3名筏工打游客被拘，当地医院：补助筏工3万，他们也要糊口

近日，广西桂林阳朔遇龙河景区筏工用船桨殴打游客的事件在网上引发热议。7月19日，阳朔县文化广电体育和旅游局发布通报称，涉事3名筏工因殴打他人已被行政拘留，对涉事公司相关领导进行停职处理。

7月21日下午，一张截图在网上流传，显示由桂林银海医院官方账号发布了“我们对遇龙河筏工事件的看法”，称三名筏工被开除后所受到的心理伤痛远大于被侮辱、殴打的肢体伤痛，医院将拿出3万元补助给三位筏工。并表示，“如果大家同意请转发，不认可请无视”。

![](https://inews.gtimg.com/om_bt/ObDxGTQ8F0x1bJnhYovafYgv8Ypbl4kSOoRratCqouvKUAA/1000)
阳朔县文化广电体育和旅游局此前通报称：2023年7月18日15点20分许，阳朔县遇龙河景区遇龙桥码头发生一起筏工与游客冲突事件。事发后，阳朔县委、县政府高度重视，迅速成立工作小组进行调查处置。经调查，事件起因系游客嬉水影响竹筏靠岸引起口角，进而引发肢体冲突。冲突发生后，已于第一时间将1名游客送至医院检查，经初步诊断身体无大碍。目前，涉事3名筏工因殴打他人已被公安机关处以行政拘留十五日并处罚款；对涉事的遇龙河公司主要领导和分管领导进行停职处理；按相关制度规定对码头管理人员进行处理。我局已向相关游客通报调查处理情况并真诚致歉。下一步，我县将举一反三，对全县旅游市场加强监管，对旅游从业人员加强教育监督，提升旅游服务质量。

![](https://inews.gtimg.com/om_bt/O5gu3uaPe3qxpov3XOBfuHTeX5GX1PibD_-LK42W7a6d4AA/1000)
网传桂林银海医院官方账号的“看法”为：“我们认为用船桨打人是不对的。我们认可阳朔县政府和公安机关对此事的处理。我们倡议筏工是底层劳动者，他们也需要工作、也需要养家糊口、也有必需的人格尊严需要保护……作为桂林本地医院我们力量绵薄，但愿为这三个筏工做力所能及之事：这三个筏工被开除后所受的心理伤痛远远大于被侮辱、殴打的肢体伤痛。为此我院将拿出三万元补助给三位筏工。以《遇龙筏工尊严补助》名义作为他们失去工作后的生活补助。”

该院官网简介显示，桂林银海医院始建于1956年，是一家具有66年历史的二级医院。

7月22日，潇湘晨报记者联系到该院办公室的一名工作人员，他表示此篇文章内容是真实的，公众号内容发布是经过审核的。至于为什么会删除，是否涉及炒作等网友质疑的问题，该工作人员表示不清楚。

潇湘晨报记者周凌如 实习生章家怡 周迎迎
