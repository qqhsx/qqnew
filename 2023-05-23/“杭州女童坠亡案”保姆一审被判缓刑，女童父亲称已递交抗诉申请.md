# “杭州女童坠亡案”保姆一审被判缓刑，女童父亲称已递交抗诉申请

5月22日，杭州女童电梯坠亡案的女童父亲在其个人社交媒体上发布视频，称其已经递交抗诉申请书。5月18日，杭州市拱墅区人民法院一审公开宣判被告人吴秀芳过失致人死亡案，被告人吴秀芳判处有期徒刑一年，缓刑二年。
