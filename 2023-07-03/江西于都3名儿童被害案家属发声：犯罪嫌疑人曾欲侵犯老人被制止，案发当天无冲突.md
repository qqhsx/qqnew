

# 江西于都3名儿童被害案家属发声：犯罪嫌疑人曾欲侵犯老人被制止，案发当天无冲突

近日，江西于都县3名儿童被害一事引发关注。7月1日，江西于都县公安局发布警情通报称，6月12日18时许，该县宽田乡发生一起致3人死亡刑事案件。当地警方于当晚20时许将犯罪嫌疑人郭某某抓获归案。经审讯，郭某某对其犯罪事实供认不讳。目前，该案件侦办工作正在依法有序进行。

7月3日，记者从当地村干部处了解到，被害儿童的家人和郭某某曾因小孩子闹矛盾发生过冲突，村里出面调解过。被害儿童的奶奶称，案发前郭某某曾企图侵犯她，被她制止；3名被害儿童的母亲张女士称，案发当天，他们与郭某某没有发生任何冲突。

7月3日，红星新闻记者联系到了3名被害儿童的母亲张女士。张女士称，她的老家在农村，家里为小孩上学在外租了住处，出租房有三室，就在自己一家住进去一年后，郭某某入住。她与孩子父亲一直在外打工，奶奶在家带着3个孩子：一个女儿8岁，一个女儿5岁，最小的一个儿子才9个月。张女士并不认识郭某某，案发当天两家人没有发生任何冲突。“郭某某趁孩子奶奶回老家给孩子拿作业画本时动的手，来回路程大约半小时。”

是何原因让犯罪嫌疑人下此毒手？被害儿童的奶奶告诉红星新闻记者，他们和郭某某租住在一套房子里，有时候孙女跑外边客厅或上楼都能碰到郭某某。奶奶称，郭某某老是抱她的两个大孙女
，孙女给她说过，郭某某抱她时会摸她隐私部位。“我要求郭某某不要抱我孙女，因为这事我们吵过，可能最大的矛盾就是这个。”

奶奶还表示，后来郭某某几次想侵犯她，被她制止。案发前几天，她带小孩进屋后没关门，郭某某就尾随进来，她让他出去，还拿小孩的裤子抽打了他。“他强行要睡我身边，我坚决拒绝了。他拿100块钱给我，我制止了他，警告他别侮辱我……”奶奶称，随后郭某某将她按到床上，她拼命踹了他肚子，郭某某这才出去了。

张女士表示，她曾听奶奶提起过郭某某对孩子动手动脚的事情，为此她还专门打电话警告对方。“谁知道郭某某会做出这样的事情来，我现在回到了老家。”张女士称，她现在住在娘家，奶奶住在相关部门安排的安置房内，相关部门还给他们家捐了款。

“我现在只想要为我们家3个孩子讨回公道。”张女士称，6月她因为办身份证回了趟老家，却没曾想到是和孩子最后一次相见。目前3个孩子还未下葬，他们也未收到来自郭某某的道歉，张女士希望司法机关能严惩凶手。

红星新闻记者从当地一名村干部处了解到，两家人曾因小孩子闹矛盾发生冲突，村里出面调解过，但是对性骚扰一事并不了解。

7月1日，江西于都县公安局发布警情通报称，6月12日18时许，该县宽田乡发生一起致3人死亡刑事案件。案发后，当地警方迅速启动命案侦办机制，组织警力开展案件侦办工作，于当晚20时许将犯罪嫌疑人郭某某抓获归案。经审讯，郭某某对其犯罪事实供认不讳。目前，该案件侦办工作正在依法有序进行。

红星新闻记者 罗丹妮
