# 香港名媛碎尸案死者前公公情妇获准保释候查 涉嫌协助罪犯

香港名媛蔡天凤碎尸案有新进展。据南都此前报道，香港警方2月26日下午拘捕了一名47岁伍姓女子，涉嫌协助罪犯。警方称其是该案中被拘捕的65岁男子、蔡天凤前夫父亲的情妇。据悉，该女子于2月28日获准保释候查，须于3月下旬向警方报到。

![](https://inews.gtimg.com/om_bt/OXgwAQi_AAblZXijpTv1lNoOLgfDhObzU8ILUz76DJHL0AA/1000)_伍姓女子及蔡天凤前夫父亲。_

南都此前报道，2月21日，香港名媛蔡天凤被发现失踪。警方调查后，认为蔡天凤前夫一家存在作案嫌疑。早前被拘捕的4人中，3名男子（蔡天凤前夫、蔡天凤前夫父亲、蔡天凤前夫哥哥）被落案暂控一项谋杀罪，一名女子（蔡天凤前夫母亲）被落案暂控一项妨碍司法公正罪。2月27日，早前被拘捕的涉案4人在九龙城裁判法院提堂。此次提堂暂无须答辩，以待警方进一步调查，包括科学鉴证及检查通话记录。裁判官将案件押后至5月8日下午再讯。

此外，香港警方26日下午在尖沙咀区拘捕了一名47岁伍姓女子，涉嫌协助罪犯。警方称其是该案中被拘捕的65岁男子、蔡天凤前夫父亲的情妇。该女子为娱乐场所工作女子，早前结识蔡天凤前夫父亲并发展成为情人关系。据港媒报道，蔡天凤残骸被发现的大埔龙尾村一村屋单位就是两人在约两周前一同租下，她又以月租4万元租赁九龙站凯旋门豪宅单位，藏匿死者前夫。

目前，伍姓女子已于2月28日获准保释候查，须于3月下旬向警方报到。
