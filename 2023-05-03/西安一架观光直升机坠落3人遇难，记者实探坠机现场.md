# 西安一架观光直升机坠落3人遇难，记者实探坠机现场

极目新闻记者 舒隆焕 丁伟

5月2日，一架小型民用直升机在西安市灞桥区狄寨原附近坠落，机上三人不幸遇难。5月3日下午，极目新闻记者在坠机现场看到，现场已清理完毕，周边树木有不少树枝被截断。

据央视新闻客户端消息，2023年5月2日14时许，陕西龙翼通用航空有限公司一架小型民用直升机在西安市灞桥区狄寨原附近坠落，机上三人不幸遇难，救援工作已经结束。善后处置工作正在开展，事故原因正在加紧调查中。

图片来源：央视新闻_

5月3日下午，极目新闻记者致电陕西一景区，工作人员称，其景区也有陕西龙翼通用航空有限公司提供的直升机观光项目，但2日下午接到通知，该观光项目暂停运营，重开时间待定。

一位曾经在陕西龙翼通用航空有限公司工作的人士向极目新闻记者透露，飞机出事后，他就得知了消息，“机上有我认识的人，我心情很不好。”

天眼查数据显示，陕西龙翼通用航空有限公司成立于2017年，位于陕西省西安市，是一家以从事航空运输业为主的企业，其经营范围包括：航空商务服务、航空运营支持服务、民用航空材料销售、民用机场运营、公共航空运输、通用航空服务、飞行签派员培训、民用航空维修人员培训、民用航空器驾驶员培训等。

3日下午4时许，极目新闻记者到达了直升机坠落地，坠落地点位于西安市灞桥区狄寨原风光路附近的一处樱桃园内，此地离白鹿仓景区有几公里的距离。记者在现场看到，现场已经清理完毕，周边树木树枝已断。据当地村民介绍，5月2日下午2时许，事故发生后，有当地村民准备前往现场，但当时道路已拉起警戒线，120、警察等相关部门工作人员已赶赴现场进行处置。“后来飞机残骸被拉走了，进来后，发现树枝断了不少，有被锯断的，也有被飞机压断的，而且现场还有祭拜的痕迹。”当地村民说。

极目新闻记者从临近的白鹿仓景区了解到，原本开设的直升机观光项目因为该起直升机坠落事件被叫停。
