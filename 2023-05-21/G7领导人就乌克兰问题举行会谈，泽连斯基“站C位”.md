# G7领导人就乌克兰问题举行会谈，泽连斯基“站C位”

当地时间5月21日，七国集团（G7）领导人在日本广岛就乌克兰问题举行工作会议。乌克兰总统泽连斯基20日飞抵广岛参加会议，在21日的会前合影中，他站在各领导人的正中。

![](https://inews.gtimg.com/om_bt/OhaH-J9AabJLoGvPJkQnRwF-O3BqwSxuFPzZvG9ioqFKoAA/1000)
据法国国际广播电台（RFI）报道，泽连斯基20日搭乘法国空军专机抵达广岛，随即与G7各国领导人开展一系列双边会谈。他在抵达广岛后发表推文称，“在这次峰会之后，和平将更近。”

在抵达峰会现场后，泽连斯基立即会见了英国首相苏纳克、意大利总理梅洛尼、德国总理朔尔茨、法国总统马克龙和印度总理莫迪等领导人。

另据半岛电视台报道，21日，泽连斯基又陆续会见了加拿大总理特鲁多、印尼总统佐科，并计划与美国总统拜登、韩国总统尹锡悦单独会面。在21日乌克兰问题工作会议开始前的一张合影中，泽连斯基站在各领导人正中。

此前，法国空军派遣一架空客A330在波兰边境接载泽连斯基，并先将其送至沙特参加阿拉伯联盟峰会，然后再将其送至日本广岛参加G7峰会。
