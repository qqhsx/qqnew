# 线上也“抢人”！千万人围观招聘直播间，投完简历3天到岗

中新经纬2月5日电
(常涛)对深圳硕腾科技总经理罗畅来说，招人，是开年返岗的头等大事。“我们新组建了跨境电商新媒体运营团队，要招二三十人，一部分做直播，一部分运营海外社交媒体。”罗畅说。

有企业招人为了直播，也有企业直播为了招人。就在各地安排专车、专列、专机开启“抢人大赛”的同时，直播带岗、直播招聘等线上“抢人”比赛，也在激烈进行。

直播带岗成开年流行动作

企业用工需求旺，务工人员返岗早，春节刚过，中国劳动力市场呈现出加速回暖“两头旺”。

近日，多地跨省招工“抢人”引发热议，不少地方专门开通护送返岗人员的专车、专列、专机等，助力务工人员复工。

“抢人”的一幕同样发生在线上直播间里。

中新经纬注意到，在不少地方，直播带岗、直播招聘成为开年的流行动作。例如1月29日，河南叶县举行了一场点对点送工直播带岗新春线上招聘会，共有10家企业参加，提供1500余个招聘岗位，初步达成就业意向300余人；2月1日，武汉新洲首场直播招聘会，吸引了超3万人观看。

事实上，一些招聘直播间过年也没有下线。中新经纬从快手了解到，1月10日至1月31日，快手快聘推出了“新春招工会”，瞄准蓝领群体就业，上线近45万个岗位，开设直播招聘近30万场。节后上班第一天，1月28日，快聘“新春招工会”单日简历投递量就突破了50万。

千万人围观，一场直播收万份简历

立臻科技是一家位于江苏省昆山市的科技制造企业，2022年9月30日，立臻科技进行了首场招聘直播，截至目前已直播了112场，保持着每天直播的频率。

据立臻科技快手账号相关负责人向中新经纬介绍，开年第一场招聘直播，也就是1月28日这场直播，共有近千万人围观，收到的简历近万份，有求职者投递完简历3天后，2月1日就已到岗入职。

新年启幕，立臻科技已将直播招聘常态化，并将每天的直播时长由3-4小时延长为6小时以上。“今年春节期间我们做的招聘直播，得到了工友们的认可，面对面的形式不仅拉近了距离，还有助于帮他们了解厂区，解决问题。”上述负责人说。

员工短缺制约生产有所缓解

新年各地上演的“抢人大赛”，一方面是因为春节过后是劳务人员外出求职的“黄金期”，招工“先人一步”，才能开局“快人一拍”；而另一方面，则是由于长期存在的企业尤其是制造企业招工难问题。

2021年国家统计局曾对9万多家规模以上工业企业进行调查，结果显示，约44%的企业反映招工难是面临的最大问题，该比例创下几年来新高。在结构性矛盾方面，沿海的工业和外贸大省，很多企业普遍反映存在招工难的问题。在当时一些调研中，很多企业反映，一线的普工难招，高技能人才难招，技术工人难招。

不过，目前这一问题有所缓解。国家统计局1月31日发布的调查显示，1月份大、中、小型企业中反映劳动力供应不足的比重均低于11%，较上月明显下降，员工短缺制约生产的情况有所缓解。

但面对各地各行业开启加速模式带动用工需求旺盛，持续促进就业、保障用工仍是重要工作。

据媒体报道，多家企业表示，劳务市场热度攀升，一方面是因近期订单增长加大了基础性岗位的招聘；另一方面是春节假期后对今年发展更有信心，提前为接下来扩大生产招揽储备人才。“去年下半年，我们拿到了一笔近2亿元大单，招人有底气也有必要，公司目前80%以上的跨境电商业务来自阿里国际站，未来还要进一步扩大。”罗畅说。

而从人才端，据智联招聘最新调研数据显示，春节后首周求职人数同比增长19.1%。

当前，由人社部等11部门联合部署的“2023年春风行动暨就业援助月”专项服务活动正在全国开展，持续到3月末，将集中为农村劳动者、就业困难人员以及用人单位进行就业帮扶，预计将为劳动者提供3000万个就业岗位。

人社部数据显示，各类招聘服务已全面发力。截至1月31日，已累计举办各类招聘活动约16000场，发布岗位近1000万个。多地人力资源市场负责人均表示，企业用工和劳动者求职预计元宵节后将迎来高潮。
