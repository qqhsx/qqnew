# 新冠已致692万人死亡，世卫宣布不再构成“国际关注的突发公卫事件”

当地时间5日，世界卫生组织总干事谭德塞在日内瓦举行的发布会上宣布，新冠疫情不再构成“国际关注的突发公共卫生事件”。世卫组织4日举行了第十五次新冠突发事件委员会会议，专家们对全球新冠疫情形势进行了评估，世卫组织总干事谭德塞根据专家们的建议做出了这一决定。

总台记者 杨春：我身后就是世卫组织总部，当地时间5日下午，世卫组织在这里宣布，新冠疫情不再构成“国际关注的突发公共卫生事件”。

2020年1月30日，世界卫生组织召开新闻发布会，宣布新冠疫情已构成“国际关注的突发公共卫生事件”。根据世卫组织的数据，截至目前，新冠疫情已在全球造成6921614人死亡。

（总台记者 易歆 杨春 李春元）
